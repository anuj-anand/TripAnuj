# Food Delivery App Front-End React JS Project

For package install:
       run -npm install;
For local run:
       npm run dev;
For built:
       npm run build;

- Smooth Scrolling
- Smooth animations and effects
- Change Menu Category
- Add to cart & Remove from cart
- View Cart
- Proceed to payment
- Website fully responsive
